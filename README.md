### AIに投げる用のプロンプトをここにメモしておく  
現在考えているアプリについて教えます．私はたくさんのアニメを見ているのですが，そのアニメを管理するアプリを作りたいです．アニメタイトルを，現在視聴中，視聴完了というステータスに分け，管理したいです．それぞれのアニメデータには，話数，放送時期，ジャンルなど複数のデータが存在します．登録画面ではアニメを登録することができ，視聴中のアニメに関してはどこまで見たかが分かるように，ボタン一つで話数を更新することができます．また，視聴完了したアニメに関しては，分析ツールを使うことができ，例えば，自分が見たアニメについて，どのジャンルを最も見ているのかを円グラフにして表示するなど，自分がこれまで見てきたアニメに関する分析ができるようになります．このようなアプリを作ろうと考えています．まずは，このアプリを完成させるまでに必要な技術や知識について教えてください
